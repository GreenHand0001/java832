# 选课管理系统简介 java832

## 项目概述

选课管理系统是一个基于前后端分离模式开发的应用，旨在为教育机构提供课程管理、选课分配、成绩录入与查询等功能。系统通过高效的流程处理，提高学校的教学管理效率。

## 技术栈

- **后端技术：**
  - Spring Boot
  - Mybatis-Plus
  - JDK 8
  - MySQL 5.7
  - Maven
- **前端技术：**
  - HTML
  - CSS
  - Vue
  - Element-UI
  - Axios
- **开发环境：**
  - IntelliJ IDEA
  - Node.js

## 功能模块

### 系统角色

- **管理员：**
  - 院系管理
  - 专业管理
  - 班级管理
  - 课程管理
  - 选课管理
  - 学生管理
  - 教师管理
  - 管理员管理

- **教师：**
  - 授课查询
  - 教师课表
  - 成绩录入

- **学生：**
  - 选课
  - 已选课程管理
  - 课表查询
  - 成绩查询
  - 信息维护

## 运行环境

- JDK 8

## 核心亮点

- **前后端分离：** 前后端分离的开发模式，提升开发效率，降低系统耦合度。
- **高效管理：** 提供全面的教学管理功能，覆盖从院系到成绩的全方位管理需求。
- **易用性强：** 界面友好，操作简便，便于不同角色的用户快速上手和使用。

## 部署步骤

1. 准备运行环境：安装JDK 8、MySQL 5.7、Node.js。
2. 构建后端项目：使用IntelliJ IDEA打开后端项目，执行Maven构建，生成可执行的JAR文件。
3. 部署后端服务：在服务器上部署JAR文件，启动后端服务。
4. 构建前端项目：进入前端项目目录，执行`npm install`安装依赖，然后执行`npm run build`构建生产环境代码。
5. 部署前端项目：将构建后的前端代码部署至Web服务器。
